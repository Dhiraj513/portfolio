# 🌐 Dhiraj Kushwaha — Portfolio

A modern and responsive **personal portfolio website** built using pure HTML, CSS, and JavaScript.
Designed to showcase skills, education, and provide a seamless way to connect.

---

## ✨ Features

* 🎨 Clean and premium UI design
* 📱 Fully responsive (mobile-friendly)
* ⚡ Smooth scrolling and animations
* 🧠 Interactive sections (About, Skills, Education)
* 📩 Working contact form using **Formspree**
* ✅ Custom success modal on form submission

---

## 🚀 Live Demo

👉 https://kushwahadhiraj.com.np/

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom styling + animations)
* JavaScript (Vanilla JS)
* Formspree (for email handling)

---

## 📁 Project Structure

```
index.html   # Main file (contains HTML, CSS, JS)
```

## ⚙️ Setup & Usage

1. Clone the repository:

```bash
git clone https://github.com/Dhiraj513/portfolio
```

2. Open `index.html` in your browser

---

## 📬 Contact Form

The contact form is integrated with **Formspree** for handling submissions.

To use your own form:

* Replace the Formspree endpoint:

```
https://formspree.io/f/xrerdjvo
```
